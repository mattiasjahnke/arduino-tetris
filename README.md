# Arduino Tetris

## Material

* 2x MAX7219 (driving 2x 8x8 LED Matrices)
* 1x Arduino Trinket (or equivalent)
* 4x momentary buttons
* 4x resistors (I used 100k ohm)
* 1x 9v battery
* Extra (perfboard, wires, soldering iron etc)
